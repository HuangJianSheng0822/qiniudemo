# 设计
我们是一个多元化的在线平台，为那些热爱潮流、音乐和游戏的用户提供了无尽的乐趣和体验。无论您是寻找新灵感、探索音乐世界，还是与其他游戏迷互动，我们都为您提供了一个充满活力和创造力的社区。加入我们，探索您热爱的领域，分享您的激情，一同塑造潮流、音乐和游戏的未来！
直播、点播弹幕系统，前端采用vue3,后端基于springboot
